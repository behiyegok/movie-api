# movie-api
NodeJS - Movie API

# Movie 
| Route | HTTP Verb | POST Body | Description |
| --- | --- | --- | --- |
| /api/movies | `GET` | Empty | List All Movies. |
| /api/movies | `POST` | { title : "foo", imdb_score , category : "bar" , country, year } | List All Record |
