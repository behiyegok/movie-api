 * git clone https://github.com/behiyegok/movie-api.git . : git clone repoyu bilgisayarımaz kopyalar cloneURL bilgileri nokta (.) bulunduğum dizine kopyala oluyor

 * git status : değişiklik yapılan dosyaları gösterir

 * git add . : bütün dosyalarımı versiyon kontrole ekle
 
 * git commit -m "açıklama ekleme" : commitin açıklaması

 * git push origin master : git reposuna gönderme

 * "start": "node ./bin/www", production da çalışma komutları 
   "start:dev": "nodemon ./bin/www", local de çalışma komutları çalıştırmak için artık : npm run start:dev
